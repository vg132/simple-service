JavaScript Proxy

When needing to call the service from JavaScript you can use the included JavaScript proxy.

Step 1

Include one of the following script in your HTML document:

<!-- minified code for production -->
<script src="$ServiceUrl$proxy.min.js" type="text/javascript"></script>

<!-- uncompressed code for development -->
<script src="$ServiceUrl$proxy.js" type="text/javascript"></script>

Step 2

// instantiate a new proxy
var service = new $ServiceName$();

// call a service method
service.myMethod('Hello', 'World', function (response) {
    if (!response.error) {
        console.log(response.value);
    }
});
